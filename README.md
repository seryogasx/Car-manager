# Car-manager

iOS приложения для автолюбителей. Учет личных авто, статистика, уведомления о необходимых работах, например, смене сезонности резины.

Стек:
Arch: MVVM + RxSwift
Storage: Realm
UI: UiKit + SnapKit (только ручная верстка), CoreGraphics (для графиков)
